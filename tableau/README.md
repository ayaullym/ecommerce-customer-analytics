# Tableau Dashboard

**Customer Churn & Loyalty Dashboard**

 **Tableau Public (Interactive):**  
https://public.tableau.com/views/CustomerChurnLoyalty/Dashboard1?:language=en-US&:sid=&:redirect=auth&:display_count=n&:origin=viz_share_link

 **Local File:**  
`customer_dashboard.twbx`

This dashboard visualizes:
- Churn risk distribution
- Customer recency patterns
- Satisfaction vs churn risk
- Churn by loyalty membership tier

The `.twbx` file allows the dashboard to be opened locally in Tableau Desktop,
while Tableau Public provides an interactive online version.
